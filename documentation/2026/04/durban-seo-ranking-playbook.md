# Durban SEO Ranking Playbook (Pro Graphics)

## Goal
Improve organic rankings and lead generation for:
- vehicle branding Durban
- sign boards Durban
- Durban signage
- fleet branding Durban
- custom signs Durban

## What was implemented in code
- Durban-targeted blog content added and optimized for local intent.
- Metadata improved on blog and service pages (title, description, keywords, canonical).
- JSON-LD schema added/expanded:
  - CollectionPage + BlogPosting on blog pages
  - BlogPosting schema on single posts
- Internal links strengthened:
  - Blog -> service pages (`/vehicle-branding`, `/sign-boards`, `/quote`)
  - Service pages -> blog resources
- Sitemap expanded to include blog hub and key posts.

## Weekly ranking workflow (30 minutes)

### 1) Google Search Console (primary source)
1. Open **Performance > Search results**.
2. Filter by:
   - Country: South Africa
   - Query contains: Durban
   - Page: `/vehicle-branding`, `/sign-boards`, `/blog/*`
3. Export data for last 28 days.
4. Track:
   - Clicks
   - Impressions
   - CTR
   - Avg Position

### 2) Ahrefs or Ubersuggest (cross-check)
Track these keywords weekly:
- vehicle branding Durban
- sign boards Durban
- Durban signage company
- fleet branding Durban
- custom sign boards Durban

Capture:
- Current position
- URL ranking
- Keyword difficulty
- Search volume trend

## Indexing + publishing checklist (for each new SEO post)
- [ ] Primary keyword in title and intro paragraph
- [ ] Secondary keyword in H2/H3
- [ ] 2-4 internal links to service/quote pages
- [ ] JSON-LD present (BlogPosting)
- [ ] Canonical set
- [ ] URL included in sitemap
- [ ] URL submitted in Google Search Console URL Inspection

## Content cadence recommendation
- Publish **2 SEO blog posts per month** focused on Durban buyer intent.
- Refresh top pages (`/vehicle-branding`, `/sign-boards`) every 6-8 weeks with:
  - Updated proof points
  - New internal links
  - Clear conversion CTA to `/quote`

## KPI targets (next 90 days)
- Top 10 for “vehicle branding Durban”
- Top 10 for “sign boards Durban”
- +30% non-branded organic clicks to `/vehicle-branding` + `/sign-boards`
- +20% quote submissions from organic landing sessions
